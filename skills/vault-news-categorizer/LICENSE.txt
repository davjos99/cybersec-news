This skill (vault-news-categorizer) is an original work built for
Agent Skills Academy Vault Module G.6 (Capstone — Content-Curation Chain).

The Threats/News/Advice taxonomy is a cybersec-specific adaptation of the
universal "act now / update mental model / file for later" triage pattern
used in editorial newsroom workflows. The pattern is widely used and not
attributable to any single source.

The deterministic categorizer (scripts/categorize.py) is a heuristic
fallback for headless orchestration (cron / GitHub Actions). When this
Skill is invoked through Claude Code, the LLM does the categorization
using the SKILL.md guidance directly.
